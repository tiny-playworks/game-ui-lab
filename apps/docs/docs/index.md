# Tiny Playworks Game UI

Motion-first React primitives for combat feedback, HUD state, rarity framing, and loot moments.

This Phase 2 site is the public entry for installation, API discovery, and the live lab.

## Start here

- [Install the packages](/guide/getting-started)
- [Browse primitives](/primitives/)
- [Review tokens](/tokens/)
- [Open the live lab](/lab/)

## Package split

- `@tiny-playworks/game-ui` exports React primitives and the shared stylesheet.
- `@tiny-playworks/tokens` exports token metadata and the CSS variable foundation.
- The gallery remains a moving lab, not the source of API truth.
