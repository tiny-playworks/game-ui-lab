# Primitives

Current public primitives:

- `DamageNumber` for floating combat text.
- `FloatingToast` for short-lived feedback.
- `ComboCounter` for encounter tempo.
- `HealthBar` for HP and shield state.
- `ResourceMeter` for mana, energy, and stamina.
- `CooldownSlot` for ability readiness.
- `StatusBadge` for buffs, debuffs, warnings, and durations.
- `LootCard`, `LootStack`, and `RewardReveal` for loot flow.
- `RarityBorder` for collectible emphasis.
- `GameUiProvider` for the token scope.
