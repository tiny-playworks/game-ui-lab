# Tokens

`@tiny-playworks/tokens` provides the CSS variable foundation used by the primitives.

```tsx
import { gameUiTokenGroups } from '@tiny-playworks/tokens';
import '@tiny-playworks/tokens/css';
```

The current token groups cover:

- Core dark surface colors.
- Combat feedback tones.
- Rarity colors.
- HUD resource and status colors.
- Motion durations and easing.
- Glow, radius, and spacing values.

The token package is intentionally small. It is not a Figma sync pipeline in Phase 2.
